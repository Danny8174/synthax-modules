Synthax Command Reference

log

Outputs a message to the console.

log "Hello, world!"

wait

Pauses execution for a number of seconds.

wait 2

set

Stores a variable for reuse.

set mood = "curious"

askAI

Queries the AI with a prompt. Use → to assign the response to a variable.

askAI "What is Synthax?" → answer

predict

Requests a prediction or suggestion. Output is assigned to a result variable.

predict "Next move?" → step
